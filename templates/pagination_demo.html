API Information
Evidence Results API
GET /api/results?page=1&per_page=10

Returns paginated evidence results with metadata including total count, current page, and navigation info.

Processing Queue API
GET /api/queue?page=1&per_page=10

Returns paginated queue items when page parameter is provided, otherwise returns queue statistics.

Response Format
{
  "success": true,
  "data": [...],
  "pagination": {
    "current_page": 1,
    "per_page": 10,
    "total_pages": 5,
    "total_items": 50,
    "has_next": true,
    "has_previous": false,
    "next_page": 2,
    "previous_page": null
  }
}
